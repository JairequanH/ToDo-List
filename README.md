# ToDo-List
Simple todo-list created using HTML, CSS, and JS.

- User can use the input box to type what item they want to add to their todo-list and can add it by either pressing [Enter] or clicking the add button.
- There must be an input in order to add an item to the list. Input cannot be null.
- Items in the list are indexed by the order they are added.
- Items can be checked, striking a line through the item to mark it as completed.
- Items can be removed from the list, automatically adjusting the list accordingly.
- The todo-list is automatically saved to local storage after every addition or removal to the list.
